# 🏹 BedWars AutoFarm

![Version](https://img.shields.io/badge/version-v1.0-blue) ![Status](https://img.shields.io/badge/status-stable-brightgreen) ![Made By](https://img.shields.io/badge/made%20by-Sxzly-purple)

A specialized and lightweight autofarm script for BedWars, designed for efficiency and stability.

## ✨ Features

* 🏆 **Win Farming:** Automatically secures wins to boost your stats.
* 🔄 **Auto Rejoin:** Automatically joins a new match after finishing, allowing for 24/7 farming.
* ⚡ **Fast Execution:** Optimized code for minimal delay.
* 💾 **Reliable:** Designed to run for long sessions without crashing.

# 📦 Installation

## Quick Load (Recommended)

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/Sxzly/BedWarsAuto/main/script.lua"))()
```
## 📦 Manual Install

1. Download **script.lua** from the repository.
2. Paste it into your executor.
3. Run it while in a **BedWars** lobby or match.

# 🎮 How to Use

1. Open **BedWars**.
2. Execute the script.
3. The script will automatically:
    * ⚔️ Start farming wins.
    * 🔄 Re-queue and join new games.
4. Track your progress in the UI!
