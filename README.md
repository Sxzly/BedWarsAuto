A specialized and lightweight autofarm script for BedWars, designed for efficiency and stability.

## ✨ Features

* 🏆 **Win Farming:** Automatically secures wins to boost your stats.
* 🔄 **Auto Rejoin:** Automatically joins a new match after finishing, allowing for 24/7 farming.
* ⚡ **Fast Execution:** Optimized code for minimal delay.
* 💾 **Reliable:** Designed to run for long sessions without crashing.
